## サービスの紹介


毎日家族のために食事を作っている女性向けに、  
「今日の食事を何にしようか？」と考える憂鬱な仕事から解放する点に特化した  
ウェブアプリケーションです。


ユーザーは、自分がよく作る料理のレパートリーを登録するだけで、  
１日分、１週間分などの期間で、アプリからおすすめの献立が紹介されます。  


## サポートしていない機能


1. レシピの情報はありません  
　　> 外部サービスへのリンク保存、メモ機能はあります。
2. カロリーの表示はありません
3. 栄養素の表示はありません
